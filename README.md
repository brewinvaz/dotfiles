# Steps to follow

This guide assumes that `zsh` is the default shell on Mac OS.

## Terminal Configuration

### iTerm2

1. Install [iTerm2](https://iterm2.com)

### Oh My Zsh + PowerLevel10k

1. Install [Oh My Zsh](https://dev.to/abdfnx/oh-my-zsh-powerlevel10k-cool-terminal-1no0)
2. Install [PowerLevel10k](https://dev.to/abdfnx/oh-my-zsh-powerlevel10k-cool-terminal-1no0)
3. Configure p10k and setup [plugins](https://dev.to/abdfnx/oh-my-zsh-powerlevel10k-cool-terminal-1no0)
4. Use `.zshrc` file in the repo for reference.

### Misc. Terminal Config

1. Fancy `ls` output [setup](https://github.com/Peltoche/lsd)
